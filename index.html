◈ ESCAPE ROOM

a series of programming challenges — pick your level

v1 — Web Security web · scripting

Automate HTTP requests to navigate a maze before the session expires. The challenge is recognising the service behaviour and writing a fast enough client.
v2 — Algorithmic graph algorithms · Python

Implement a pathfinding agent in Python — Dijkstra, A*, weighted A* — that navigates a procedurally generated maze while avoiding randomly moving monsters. Runs entirely in the browser.
☠ The Trap Room — Minimax game theory · minimax · Python

You and the Monster take turns removing 1–3 coins from piles of cursed gold. Whoever takes the last coin triggers the trap and dies. Implement the Monster's optimal strategy using minimax — then try to beat it.
♞ Chess Arena — Multi-Agent multi-agent · pathfinding · Python

Knight-move maze escape. Write a Python agent that navigates as a chess knight, avoids or eliminates rivals, and reaches the exit first. Pit multiple agents against each other — and against the bots.
⬡ Neural Trap Room — Deep Learning neural networks · transformers · Python

Three neural architectures learn to play misère Nim from game data. Explore how a one-hot lookup table, a Grundy-aware scalar network, and a causal transformer each discover game theory from scratch — and where they fail.
