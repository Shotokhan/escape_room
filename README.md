# escape_room
<br>
Original web security challenge: See challenge description in src/problem_statement.

This challenge has evolved to a series of challenge, it started as a simple problem aimed at scripting with Python requests library and finding a flaw in a problem generation algorithm, then it became a way to explore DSA, including search algorithms, then heuristics and then AI algorithms such as multi-agent (e.g., Minimax).

It then uses one of the problems as example for deep neural network training, comparing from a practical and theoretical point of view three different architectures.
